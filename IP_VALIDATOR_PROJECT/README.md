# IP Validator Project

## Description
This is a Python project that validates whether a user-entered IPv4 address is valid or invalid.

## Features
- Accepts user input
- Checks if the IP has 4 parts
- Verifies each part is numeric
- Ensures each number is between 0 and 255
- Displays whether the IP address is valid or invalid

## Technologies Used
- Python 3
- VS Code

## How to Run
1. Open the project in VS Code.
2. Open the terminal.
3. Run:
   python ip_validator.py

## Example

Input:
192.168.1.1

Output:
Valid IPv4 Address
