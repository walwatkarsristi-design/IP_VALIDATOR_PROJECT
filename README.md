# IP Validator Project

## Description
This Python project validates whether a user-entered IPv4 address is valid or invalid.

## Features
- Validates IPv4 addresses
- Checks each octet is between 0 and 255
- Displays whether the IP address is valid or invalid

## Technologies Used
- Python 3
